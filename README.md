# bspwm


These are the dotfiles for my ArchLinux bspwm rice...

- Settings for:
	
	- oh-my-zsh (shell)
	- bspwm (window manager)
	- polybar (status bar)
	- sxhkd (general key binder)
	- thunar (file manager)
	- VSCodium
	- rofi
	- jgmenu
	- other stuff like blurwal,wpgtk and more, etc.
