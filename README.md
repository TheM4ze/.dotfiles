                                         Welcome To My Dotfiles



Introduction
------------

This repo contains config/scripts for my bspwm desktop

	
	- oh-my-zsh (shell)
	- bspwm (window manager)
	- polybar (status bar)
	- sxhkd (general key binder)
	- thunar (file manager)
	- VSCodium
	- rofi
	- jgmenu
	- other stuff like blurwal,wpgtk and more, etc.
